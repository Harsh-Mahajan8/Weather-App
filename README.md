🌤 Weather App
A simple and responsive weather application built using React.js. This app fetches real-time weather data and provides users with current weather conditions, temperature, humidity, and more.

🔹 Features
✅ Search weather by city name
✅ Displays temperature, humidity, wind speed, and weather conditions
✅ Clean and user-friendly UI
✅ Deployed on Vercel
Link:https://weather-app-seven-flame-38.vercel.app/
